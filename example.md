macOS 를 사용중이시라면 아래 명령어로 모든 것들을 해결하실 수 있습니다.

```shell
brew install node && brew install yarn && brew install --cask visual-studio-code && brew install git
```

⚠️ macOS 사용자 분들중에 homebrew를 모르신다면 [여기](https://subicura.com/mac/)를 참고해주세요!
